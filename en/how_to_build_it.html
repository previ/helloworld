Chassis and motors

Several versions available, depending on material, motor type.

CoderBot has been built in several versions for differente kind of motor (dc, servo) and chassis material.

Hardware (electronic)

Raspberry Pi, motor control board, battery

CoderBot is based on Raspberry PI, the camera module is required for video streaming.

Software

Software components run on the Raspbian linux distribution.

The software components are available to download.
